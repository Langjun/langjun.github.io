applications of my macbook pro

发表于 2017-07-10 更新于 2020-03-01

分享一下自己常用的几款在Mac上的软件，真的可以提高工作效率。

编辑器

  • sublime text
    • 非常喜欢的一款简洁、轻量的编辑器，现在基本用来做临时的文本编辑等
  • intellij idea
    • 日常写码利器、各种方便，用了便知道。这款软件使用了校园的企业邮箱账号可以申请到免费使用的license, 每次申请可使用期限为一年，第二年需要更新一次。
  • Visual Studio Code
    • MS大大推出的神器，个人使用感觉是介于sublime与idea之前的一款编辑器，编写命令行操作频繁的简单demo时会用到。关键是免费。

版本控制

  • Git Tower
    • git的一个超级棒可视化工具，无论是界面的样式以及功能上都非常满足我现在使用的场景，非常方便。
  • Svn Cornerstore
    • SVN的一个可视化工具，早期参与使用svn做版本控制的项目时这是一款利器，体验最出色的一点个人觉着是在merge code的操作上。

      其他

  • Dash
    • 查文档利器，不过使用场景很少，大部分API查询均在网络搜索下进行
  • Alfred
    • 不用多说，又是一款可以节省很多时间的神器，极力推荐。
  • Paste
    • 一款剪切板的扩展工具，粗略统计，日均使用次数10+。
  • Moon
    • 用来做应用窗口快速布局、缩放等操作。用起来也会节省小部分时间。
  • iPic
    • mac上一款好用的图床软件，使用markdown写文章的场景特好用，只可惜是订阅式付费。暂时使用免费功能，上传微博图床。
  • 1Passwrod
    • 密码管理工具，用来管理自己各种站点的密码，也是很常用的工具。尤其是对于各种需要复杂密码的场景，可以随机生成密码串。
  • iStat Menus
    • 虽然它包含多种功能，包括cpu&gpu、memory、disks等等的功能，但最常用的是在状态栏点击一下就可以看到本机ip，点击ip地址可以快速复制，6.X更新后更加符合个人使用习惯及不讨厌的UI。
  • Amphetamine
    • 一款专治突然黑屏的尴尬场景。
  • PopClip
    • 一个对高亮本文的快捷操作工具条，其中可扩展功能非常丰富，最常用的搜索、翻译、拷贝等场景。
  • NTFS for Mac OS X
    • 用来兼容移动硬盘在macOS上进行读写操作
  • Bartender
    • 好用的状态栏图标管理工具，将不常用的图标隐藏起来。使状态栏干净整洁。
  • Charles
    • 目前使用过最好用的抓包工具、其中包含了https抓包、慢网模拟等等常用的功能
